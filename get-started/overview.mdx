Agent 37 gives you a hosted OpenClaw environment so you can create a bot, configure a model, connect real channels, and test everything from one dashboard without managing the infrastructure yourself.

<Info>
  **Use this section if you are new to Agent 37**:
  - Start with bot creation
  - Open the dashboard and terminal
  - Connect your first channel
  - Continue into model setup or Tailscale when you are ready
</Info>

## Recommended setup flow

<Steps>
<Step title="Create an instance in Agent 37">
  Create your first bot from the [Agent 37 dashboard](https://www.agent37.com/dashboard).
</Step>

<Step title="Open the terminal">
  Each instance includes a terminal where you can run OpenClaw commands such as `openclaw configure`.
</Step>

<Step title="Connect a channel">
  Use the channel setup flow to link WhatsApp, Telegram, Slack, or Discord.
</Step>

<Step title="Configure your model">
  Set up OpenAI, Claude, or a custom model provider depending on your workflow.
</Step>

<Step title="Test the bot">
  Send a real message through the connected platform or use Web Chat to confirm your bot responds.
</Step>
</Steps>
