## What the setup flow looks like

<Steps>
<Step title="Create an instance in Agent 37">
  Create your first bot from the [Agent 37 dashboard](https://www.agent37.com/dashboard).
</Step>

<Step title="Open the terminal">
  Each instance includes a terminal where you can run OpenClaw commands such as `openclaw configure`.
</Step>

<Step title="Connect a channel">
  Use the channel setup flow to link WhatsApp, Telegram, Slack, Discord, or another supported platform.
</Step>

<Step title="Send a test message">
  After setup, send a real message through the connected platform and confirm your bot responds.
</Step>
</Steps>

<Tip>
  If you want the fastest path, start with
  [Create your first bot](/get-started/create-first-bot), then move to

</Tip>
