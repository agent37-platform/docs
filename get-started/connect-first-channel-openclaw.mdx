After you create a bot, the next step is to connect a channel so OpenClaw can receive and send messages through a real platform.

## How channel setup works

You start in the Agent 37 dashboard, open your bot's terminal, and run the OpenClaw configuration flow. The terminal guides you through the rest of the setup for the specific platform you choose.

<Steps>
<Step title="Open your bot in the Agent 37 dashboard">
  Go to the [Agent 37 dashboard](https://www.agent37.com/dashboard), then open the instance you want to configure.
</Step>

<Step title="Open the terminal">
  Click **Terminal** for that bot so you can run OpenClaw commands inside the hosted environment.
</Step>

<Step title="Start configuration">
  Run:

  ```bash
  openclaw configure
  ```
</Step>

<Step title="Choose the channel flow">
  In the interactive menu:

  1. Select `Channel`
  2. Choose `Configure / Link`
  3. Select the platform you want to connect
</Step>

<Step title="Complete provider authentication">
  Follow the prompts for the channel you selected. Depending on the platform, this may require a QR code scan, a bot token, an OAuth flow, or account approval.
</Step>

<Step title="Send a test message">
  When setup is complete, send a real message through the platform and confirm that OpenClaw receives and responds to it.
</Step>
</Steps>
